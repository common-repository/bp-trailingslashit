Add "/" at any buddypress permalink's end but specific logged in users specific actions.

== Description ==

Look at the end of any buddypress URI if a slash is displayed and if it is not, redirect permanently non loggedin users to one with. Better for share and especialy done for crawlers.

== Installation ==

The plugin is easy to install, copy it to your wordpress's plugins path, and active it at the admin console, it will be work well.
